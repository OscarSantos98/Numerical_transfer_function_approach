# Numerical_transfer_function_approach
This repository show how to approach First-Order Linear Time Systems (LTI) with numerical techniques, additionally the algorithm proposes gains for a PID Controller.

# Pre-requisites
## Packages
- Matplotlib
- Numpy
- Sympy
- Scipy
- Pandas
